# Repository Rules

This file defines rules specific to this repository.

- Team-wide collaboration rules are stored under `docs/rules/`.
- If team-wide rules conflict with repository-specific rules, this file takes precedence.
- This repository is used to maintain collaboration rules for the Agent team.
- All documentation in this repository must be written in English.
- Development in this repository does not use branch-based merging. All changes are committed directly to `main`.
